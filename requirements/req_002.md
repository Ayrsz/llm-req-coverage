# req_002 — Validação de força de senha

## Descrição

O sistema deve validar se uma senha atende a todas as regras mínimas de
segurança. A função retorna `True` apenas quando **todas** as regras são
satisfeitas, e `False` caso contrário.

## Interface

- **Módulo a importar:** `solution`
- **Assinatura:** `validate_password(password: str) -> bool`

```python
from solution import validate_password
```

## Domínio de entrada

- `password`: string. Pode ser vazia. Pode conter qualquer caractere ASCII
  imprimível (letras, dígitos, símbolos, espaço).

## Regras

A senha é **válida** (`True`) somente se satisfizer **todas** as condições:

1. comprimento de **no mínimo 8 caracteres** (`len(password) >= 8`);
2. contém ao menos **uma letra maiúscula** (`A`–`Z`);
3. contém ao menos **uma letra minúscula** (`a`–`z`);
4. contém ao menos **um dígito** (`0`–`9`);
5. contém ao menos **um símbolo** (caractere que não é letra nem dígito; ex.:
   `!@#$%^&*()-_+=`).

Se qualquer regra falhar, o retorno é `False`.

## Classes de equivalência

- senha válida (atende às 5 regras) → `True`;
- curta demais (< 8) → `False`;
- sem maiúscula → `False`;
- sem minúscula → `False`;
- sem dígito → `False`;
- sem símbolo → `False`;
- string vazia → `False`.

## Valores-limite

- exatamente 8 caracteres válidos → `True`;
- 7 caracteres (senão tudo certo) → `False`.

## Critérios de aceitação

- AC1. `validate_password("Abcdef1!") is True` (8 chars, atende tudo).
- AC2. `validate_password("Abc1!") is False` (curta).
- AC3. `validate_password("abcdef1!") is False` (sem maiúscula).
- AC4. `validate_password("ABCDEF1!") is False` (sem minúscula).
- AC5. `validate_password("Abcdefg!") is False` (sem dígito).
- AC6. `validate_password("Abcdefg1") is False` (sem símbolo).
- AC7. `validate_password("") is False` (vazia).
