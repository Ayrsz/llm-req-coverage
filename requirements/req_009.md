# req_009 — Diferença em dias entre duas datas ISO

## Descrição

O sistema deve calcular o número de dias decorridos de uma data `start` até uma
data `end`, ambas no formato ISO `"YYYY-MM-DD"` e recebidas **como argumento**
(nunca usa a data atual). O resultado é **com sinal**: positivo quando `end` é
posterior a `start`, negativo quando `end` é anterior, e `0` quando são iguais.

## Interface

- **Módulo a importar:** `solution`
- **Assinatura:** `days_between(start: str, end: str) -> int`

```python
from solution import days_between
```

## Domínio de entrada

- `start`, `end`: strings de data no formato ISO `"YYYY-MM-DD"`.
  São parseadas com `datetime.date.fromisoformat`.
- Strings que não estejam nesse formato (ou com data inexistente, p.ex.
  `"2021-02-30"`) são **entradas inválidas**.

## Regras

1. O resultado é `(end - start).days`, ou seja, com **sinal**:
   - `end > start` → inteiro positivo;
   - `end < start` → inteiro negativo;
   - `end == start` → `0`.
2. Se `start` **ou** `end` não puderem ser parseados como data ISO válida,
   a função retorna o sentinela inteiro **`-999999`** (decisão documentada:
   preserva o tipo de retorno `int`, é determinístico e improvável como
   diferença real de dias dentro do domínio de uso). A função não levanta exceção.

## Classes de equivalência

- mesma data → `0`;
- `end` depois de `start` → positivo;
- `end` antes de `start` → negativo;
- formato/data inválido em qualquer argumento → `-999999`.

## Valores-limite

- `days_between("2021-01-01", "2021-01-01") == 0` (mesma data);
- `days_between("2021-01-01", "2021-01-02") == 1` (um dia);
- `days_between("2021-01-02", "2021-01-01") == -1` (um dia, sinal negativo);
- ano bissexto: `days_between("2020-02-28", "2020-03-01") == 2`
  (29/02/2020 existe → 28→29→01 são 2 dias);
- ano não bissexto: `days_between("2019-02-28", "2019-03-01") == 1`;
- ano completo bissexto: `days_between("2020-01-01", "2021-01-01") == 366`;
- ano completo não bissexto: `days_between("2019-01-01", "2020-01-01") == 365`.

## Entradas inválidas

- Formato fora de `"YYYY-MM-DD"` (ex.: `"01/01/2021"`, `"2021-1-1"`, `""`).
- Data inexistente (ex.: `"2021-02-30"`, `"2021-13-01"`).
- Em todos esses casos a função retorna `-999999` (não levanta exceção).

## Invariantes

- A saída é sempre um `int`.
- `days_between(a, b) == -days_between(b, a)` para `a` e `b` válidos.
- `days_between(a, a) == 0` para `a` válido.

## Critérios de aceitação

- AC1. `days_between("2021-01-01", "2021-01-01") == 0` (mesma data).
- AC2. `days_between("2021-01-01", "2021-01-02") == 1` (diferença positiva).
- AC3. `days_between("2021-01-02", "2021-01-01") == -1` (diferença negativa/sinal).
- AC4. `days_between("2020-02-28", "2020-03-01") == 2` (atravessa 29/02 bissexto).
- AC5. `days_between("2019-01-01", "2020-01-01") == 365` (ano não bissexto).
- AC6. `days_between("2021-02-30", "2021-01-01") == -999999` (data inexistente → sentinela).
- AC7. `days_between("01/01/2021", "2021-01-01") == -999999` (formato inválido → sentinela).
