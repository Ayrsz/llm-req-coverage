# req_003 — Classificação de faixa etária

## Descrição

O sistema deve classificar uma pessoa em uma faixa etária a partir da idade em
anos completos.

## Interface

- **Módulo a importar:** `solution`
- **Assinatura:** `age_bracket(age: int) -> str`

```python
from solution import age_bracket
```

## Domínio de entrada

- `age`: número inteiro de anos completos. Pode ser negativo (entrada inválida).

## Regras

Retorna uma destas strings, conforme a idade:

1. `age < 0` → `"invalido"` (entrada inválida);
2. `0 <= age <= 11` → `"crianca"`;
3. `12 <= age <= 17` → `"adolescente"`;
4. `18 <= age <= 59` → `"adulto"`;
5. `age >= 60` → `"idoso"`.

## Classes de equivalência

- idade negativa → `"invalido"`;
- criança (0–11);
- adolescente (12–17);
- adulto (18–59);
- idoso (60+).

## Valores-limite

- `age = -1` → `"invalido"`;
- `age = 0` → `"crianca"`;
- `age = 11` → `"crianca"`; `age = 12` → `"adolescente"`;
- `age = 17` → `"adolescente"`; `age = 18` → `"adulto"`;
- `age = 59` → `"adulto"`; `age = 60` → `"idoso"`.

## Invariantes

- A saída é sempre uma das cinco strings acima, exatamente nesse texto
  (minúsculas, sem acento).

## Critérios de aceitação

- AC1. `age_bracket(-1) == "invalido"`.
- AC2. `age_bracket(0) == "crianca"` e `age_bracket(11) == "crianca"`.
- AC3. `age_bracket(12) == "adolescente"` e `age_bracket(17) == "adolescente"`.
- AC4. `age_bracket(18) == "adulto"` e `age_bracket(59) == "adulto"`.
- AC5. `age_bracket(60) == "idoso"`.
